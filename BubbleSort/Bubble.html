How does bubble sort work?

Bubble sort works by looping through every value in the list and comparing it the value after it (except for the last one since there is no value after it). The current value in the list being used is the key. If the key is greater than the value after it then the swap positions. After it gets through all of the values if it swapped any values then it goes back to the beginning and loops through all the values again. Otherwise it finishes as all the numbers have been sorted.
