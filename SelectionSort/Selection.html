How does selection sort work?

Selection sort works by looping through every value (except the last), this is the original key. It then compares it to the value in the next index. If the value is less than the key value it sets the key value to the value being compared. If not then it does back to the step were it compares the key to the value of the next index. It does this until it reaches the final value. Once that happens it switches the values in the original key’s index and the final key’s index (the lowest number found). It then goes back to the beginning now using the value in the next index. Once it has looped through every value (it has reached the final index) it is done.
